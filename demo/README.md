## 🚀 快速开始

### 1️⃣ 打开演示
```
双击 demo.html
```

### 2️⃣ 演示流程（5分钟）

1. **配置参数** - 左侧面板选择各项参数
2. **点击"计算"** - 运行仿真，查看PT曲线
3. **上传数据** - 切换到"实际数据存储"标签
4. **对比分析** - 切换到"PT曲线对比"标签
5. **展示功能** - 点击各个导航菜单

### 3️⃣ 所有功能都可点击！

✅ 所有下拉菜单
✅ 所有输入框
✅ 所有按钮
✅ 所有标签页
✅ 交互式图表（缩放、平移、悬停）
