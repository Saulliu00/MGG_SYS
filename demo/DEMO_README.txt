📁 演示文件：demo.html

🚀 如何打开：
   1. 双击 demo.html 文件
   2. 或者拖拽到浏览器窗口
   3. 或者右键 → 打开方式 → 选择浏览器

💡 演示内容：
   ✅ 完整的仿真计算界面
   ✅ 所有参数配置可交互
   ✅ 实时PT曲线图表
   ✅ 数据对比分析
   ✅ 所有按钮都可点击

🎯 演示流程：
   1. 配置左侧测试参数（选择各项参数）
   2. 填写中间测试信息（操作员、名称、备注）
   3. 点击"计算"按钮 → 查看PT曲线
   4. 切换到"实际数据存储"标签 → 点击"上传并存储"
   5. 切换到"PT曲线对比"标签 → 查看对比图表

✨ 特色功能：
   - 交互式图表（可缩放、平移、悬停查看数据）
   - 实时通知提示
   - 三标签页设计
   - 专业UI界面

📌 注意：
   - 需要网络连接（加载在线资源）
   - 推荐使用Chrome/Edge/Firefox浏览器
   - 这是纯UI演示，数据不会保存

📖 详细演示指南：请查看 DEMO_GUIDE.md
